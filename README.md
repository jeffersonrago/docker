# Criando Lab Docker

## Server
Ubuntu 18.04.1 LTS

## Kernel
4.15.0-39-generic

## Instalando Docker
```
curl -fsSL https://get.docker.com/ | sh
```

## Versão instalada

```
docker --version
```

Resultado:
Docker version 18.09.0, build 4d60db4

## Adicionando usuário grupo DOCKER

```
sudo usermod -aG docker USER_NAME
```
